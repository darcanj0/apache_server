## Instalação

Clone o projeto

```bash
  git clone https://github.com/darcanj0/apache_server.git
  cd ./apache_server
```

Dê a permissão para rodar o script
```bash 
  sudo chmod 777 ./main.sh
```

Faça login como usuário root

```bash 
  su
```

Rode o comando
```bash
  ./main.sh
```
